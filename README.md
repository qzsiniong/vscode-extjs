# vscode-extjs README

一个提高Extjs开发效率的vscode插件

## Features

* `xtype` 验证，确保在requires中引入
* `xtype` 跳转，跳转到对应的源文件
* and so on...

## Requirements

在项目根目录下使用extjs.conf.json文件配置

```json
{
    "extjsDir":"extjs-projects",
    "extjsBase":"erp"
}
```
